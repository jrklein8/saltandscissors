# Salt & Scissors Creative Co.

Website for Salt & Scissors Creative Company — curated craft experiences
(sensory sundaes, charm bar, canvas & patch parties) in Wilmington, NC + beyond.

- `index.html` — the live site
- `classic.html` — earlier design variant, kept for comparison
- `assets/img/` — optimized photos and logos

Static site, no build step. Deployed via GitHub Pages.
